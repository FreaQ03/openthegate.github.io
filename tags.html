Basic Tags of HTML

The first tag is the HTML Tag this should always be on the start or the top of the notepad


To close a tag, you must add a slash (/) to it. If you want to close html tag, it must be written like this, /html.


The visible part of the HTML document is between and .


The paragraph,

and

tag is used to make paragraphs.


The header tag,

and

is used to make a header.


You can make a header different sizes, 1 is the largest and 6 is the smallest.
